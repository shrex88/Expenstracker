# EXPENSTRAKER
Issue Management System
Problems Solved
Manual tracking of expenses in notebooks or spreadsheets.
Difficulty in monitoring monthly spending patterns.
Lack of categorized expense management.
Budget overspending without notifications.
Time-consuming financial report generation.
Limited visibility into spending trends.
Solution Provided
The Expense Tracker automates expense recording, categorization, budget tracking, reporting, and analytics through a centralized web application.
