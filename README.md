# EXPENSTRAKER
# 💰 Expenstrack - Expense Management System

A modern web-based Expense Management System developed using Java, JSP, Servlets, and MySQL. The application enables users to efficiently track expenses, manage budgets, analyze spending patterns, and generate financial reports through an interactive dashboard.

## 📌 Project Highlights

* Secure user authentication and session management.
* Complete CRUD operations for expense management.
* Category-wise expense tracking and organization.
* Interactive dashboard with spending analytics.
* Budget monitoring and financial insights.
* PDF report generation with filtering options.
* Responsive dark-themed user interface.
* MVC-based architecture with DAO design pattern.
* MySQL database integration for persistent data storage.

## 🛠️ Issue Management System

### Problem Statement

Managing personal expenses manually can be difficult and time-consuming. Users often struggle to:

* Track daily expenses efficiently.
* Monitor monthly spending habits.
* Stay within budget limits.
* Analyze spending trends.
* Generate financial reports.

### Solution

Expenstrack provides a centralized platform where users can:

* Record and manage expenses digitally.
* Categorize transactions.
* Monitor budget utilization.
* Visualize spending patterns.
* Generate downloadable financial reports.
